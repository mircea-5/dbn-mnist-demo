About this project

Author: Yerbol Kuttybay

This is an interactive demo of a Deep Belief Network (DBN) trained on the MNIST handwritten-digit dataset. A DBN is a generative probabilistic model built by stacking multiple Restricted Boltzmann Machines (RBMs). It learns to both recognize patterns (bottom-up) and generate plausible examples (top-down).

Input (784) → RBM 0 · Hidden (500) → RBM 1 · Hidden (500) → Top RBM (2000) + Softmax (10)
Training

Greedy layer-wise pre-training on 60 000 MNIST samples. Each RBM is trained with Contrastive Divergence (CD-1), then the top layer jointly models the hidden representation and class label via a softmax unit.

How to use

Draw a digit (0–9) in the canvas, then press Run ↑ to classify it. To generate a digit, click any of the softmax units on the right side of the canvas — the network will hallucinate a matching digit.

Recognition pass ↑

Pixel activations propagate upward through two hidden layers into the top RBM. Connections light up in cyan. Gibbs sampling refines the top-layer state and drives the softmax prediction.

Generative pass ↓

After Gibbs sampling on the top RBM, the network reconstructs pixels by propagating activations downward. Connections light up in pink. The result is rendered into the drawing pad.
